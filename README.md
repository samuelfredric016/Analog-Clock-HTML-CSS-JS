 Analog Clock using HTML, CSS & JavaScript

This project is a fully responsive Analog Clock built using HTML, CSS, and JavaScript. It displays the current time in real-time with smoothly moving hour, minute, and second hands. The clock features a modern user interface with a beautiful background image, a glassmorphism-inspired clock face, responsive design, and a digital clock with the current date displayed below the analog clock.

The project demonstrates the use of HTML for page structure, CSS for styling, animations, and responsive layouts, and JavaScript for real-time clock functionality using the `Date` object. The interface automatically adapts to different screen sizes, making it suitable for desktops, tablets, and mobile devices.

Features

* 🕒 Real-time analog clock
* ⏰ Digital clock display
* 📅 Current date display
* 📱 Fully responsive design
* 🖼️ Attractive background image
* ⚡ Smooth second-hand animation
* 🎯 Clean and well-organized code
* 🌐 Cross-browser compatible

Technologies Used

* HTML5 – Structure and layout
* CSS3 – Styling, animations, and responsive design
* JavaScript (ES6) – Real-time clock functionality
